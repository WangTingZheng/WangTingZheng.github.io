manim📊

manim是著名数学科普频道3Blue1Brown使用的视频制作引擎，以前考期的时候在b站看过他的视频，当时感觉这个动画就很不一般，好奇是如何制作的，Github就给我推这个项目，目前这个项目已经有了14万的收藏量，还没使用过，估计是一个类似与画板的软件，专门对各种数学符号进行了优化，这年头的科普up主越来越硬核了😂。

tqdm🔧

一个为python编写的进度条插件，我在写爬虫pianyuan的过程中，也使用了进度条，之前的进度条是从网上抄的，抄的时候就发现很难改，在自己电脑上调试了很多遍，而且改完之后，我回过头来想把它封装成一个函数又令我头疼，看到这个工具，我眼前一亮。

musicbox🎼

一款国人编写的命令行版网易云音乐客户端，使用 NeteaseCloudMusicApi封装而来，很装逼，我很好奇它的命令行选择的效果是如何做出来的。还有什么比在命令行环境下运行网易云音乐更酷的事呢？当然有，那就是把网易云音乐运行在单片机上！

calibre📚

市面上最强大的电子书管理软件，没有之一！能像图书馆一样管理电子书，很方便地进行图书格式的转换、信息的编辑、电子书的发送（kindle），除此之外，calibre还具有强大的插件支持，官方文档很充实（但是我还是看不懂:-( ）

Real-Time-Voice-Cloning🎤

这个就很牛了，一个深度学习项目，你输入5秒你的声音，软件就会模拟你的声音来读出各种声音，但是目前只支持英文，其实多语言适配的呼声已经很强烈了，但是作者表示主要难度在于高质量的、长时间的语音文本。

xunlei-lixian📲

迅雷离线API，用于各种下载，不得不说，迅雷的会员下载速度还是很快的，这个API给了开发者无限可能，只要有了它，你就可以像迅雷本雷一样制作属于你的迅雷客户端，我最近想做一个电影搜索+电影高速下载的工具，所以就找到了它，但是这个项目早在2016年就停止更新了，再加上迅雷的离线下载服务好像关闭了，不知道这个api还能不能用，我也没试。

chrome_offine_game_on_esp8266🎮

一款在esp8266的oled屏幕上运行的谷歌chrome小游戏——恐龙跳仙人掌，在b站上看到的，地址在这儿把esp8266做成电子游戏机，在单片机这种低速度的平台开发复古游戏一直是我的梦想，有时间试一试，硬件我都有。

dm_control👩‍💻

一个深度学习项目，功能是自动控制倒立摆做出一些动作，之前准备电赛的时候看到的，之前有看到一个视频是有一个日本科学家，到公园随便捡了一根树枝，会实验室后就把它的形状扫描进计算机，再输入到数学模型中训练，按照出来的结果给树枝安装点击，数组就能像蚯蚓一样向前运动，之前还有一个游戏是讲进化的，就是你输入一些初始的几何结构，然后它会不断地在行走中改进。

platformio-core💻

这个是新一代跨平台物联网开发工具PlatformIO的核心，我是PlatformIO的忠实用户，我刚接触它的时候还想对它的核进行一些开发，现在开来我是太天真了。PlatformIO可以运行在vscode和atom上，支持各种开发板，Arduino、STM32、ESP8266等常见的开发板都有，debug、编译之类的功能也都有，这种级别的IDE应该和keil一样作为专业软件存在，但是PlatformIO是开源且免费的！（加一嘴，这家伙配合vscode使用真的是太舒服了)

PySimpleGUI📟

在被PyQt折磨了半天的我决定找一个更简单的GUI框架，于是我找到了它，以下是它的一个简单的例子：

xxxxxxxxxx
import PySimpleGUI as sg
layout = [  [sg.Text('Some text on Row 1')],
            [sg.Text('Enter something on Row 2'),                        sg.InputText()],
            [sg.Button('Ok'), sg.Button('Cancel')] ]
window = sg.Window('Window Title', layout)
while True:
    event, values = window.read()
    if event in (None, 'Cancel'):
        break
    print('You entered ', values[0])
window.close()

tinyDB📁

看到这个项目的时候我是羡慕嫉妒恨，因为我最近刚有一个想法，想要以JSON的形式储存数据，看到这个项目的时候才发现已经有人做得很完整了，文档都已经很齐全，上手很简单：

先从pip安装tinyDB

xxxxxxxxxx
pip install tinydb

引入包，向本地的db.json里插入一段字典

xxxxxxxxxx
>>> from tinydb import TinyDB, Query
>>> db = TinyDB('db.json')
>>> db.insert({'type': 'apple', 'count': 7})
>>> db.insert({'type': 'peach', 'count': 3})

我测试的时候发现tinyDB在存储中文的时候，是采用unicode的形式储存的，但是通过db.all()输出的时候，会重新变成中文。
